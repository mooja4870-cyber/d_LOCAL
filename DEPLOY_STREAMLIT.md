# Streamlit Cloud 배포 가이드 (`d_LOCAL`)

## 1. 개요
본 프로젝트(`d_LOCAL`)는 지자체·공공기관 AI/숏폼 공모전 및 지역 살아보기(체류) 지원사업의 실시간 정보를 제공하는 웹 서비스입니다.
Render 배포 대신 **Streamlit Community Cloud (`https://share.streamlit.io`)** 를 통해 배포 및 구동됩니다.

## 2. 배포 링크
- 1-Click 자동 배포 링크: [Streamlit Cloud 원클릭 자동 배포](https://share.streamlit.io/deploy?repository=mooja4870-cyber/d_LOCAL&branch=main&mainModule=streamlit_app.py)
- 배포 대상 GitHub Repository: `mooja4870-cyber/d_LOCAL`
- Branch: `main`
- Main file path: `streamlit_app.py`

## 3. 로컬 실행 방법
```bash
streamlit run streamlit_app.py
```
